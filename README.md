# Extreme (Back-end)
날씨정보를 이용한 야외 액티비티 예약 애플리케이션

## 팀 구성
| 김강민 | 김은서 |
|---|---|
|[kkmin223](https://github.com/kkmin223)|[EunSeo119](https://github.com/EunSeo119)|


## 주요 기능
- 액티비티 상품 정보 제공
- 상품 상세 정보 제공(리뷰, 상품 상세 정보)
- 검색 기능 
- 날씨 정보를 이용한 일정 예약 기능
- 마이페이지(주문 내역)
- 날씨 지도(전국 날씨 정보 제공)

## Stack
- Node.js
- MySQL
- Heroku
- Amazon S3
